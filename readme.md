*id*

star identification using convolutional and recurrent networks, triangular structure, etc.

*libstarid*

interface cpp eigen and python numpy.

*stars*

star images from pointing vectors. two coordinates are independent and expressed in a plane tangent to the sky and its image in a sensor. shift the image to center a star and identify it.
